📘 Türkçe Terminal Kılavuzu
============================

Bu sistem, terminal komutlarını Türkçeleştirerek günlük işlemleri daha anlaşılır ve akılda kalıcı hale getirir.
Aşağıda kullanabileceğiniz komutlar ve anlamları listelenmiştir:

┌───────────────────────────────┬────────────────────────────────────────────┬───────────────────────────────────────┐
│ Türkçe Komut                 │ Açıklama                                   │ Orijinal Komut / Parametre            │
├───────────────────────────────┼────────────────────────────────────────────┼───────────────────────────────────────┤
│ gir klasör                   │ Belirtilen klasöre geçiş yapar             │ cd klasör                             │
│ nerdeyim                     │ Şu anki dizini gösterir                    │ pwd                                   │
│ kimimben                     │ Kullanıcı adını yazdırır                   │ whoami                                │
│ kiminle                      │ Kullanıcının grup bilgilerini gösterir     │ groups                                │
│ klasöroluştur yeni_klasör    │ Yeni klasör oluşturur                      │ mkdir yeni_klasör                     │
│ klasörsil hedef              │ Boş klasörü siler                          │ rmdir hedef                           │
│ sil dosya                    │ Dosyayı siler (onaylı)                     │ rm -i dosya                           │
│ silr klasör                  │ Dolu klasörü ve içeriğini siler            │ rm -r klasör                          │
│ kopyala kaynak hedef         │ Dosyayı veya klasörü kopyalar              │ cp kaynak hedef                       │
│ taşı kaynak hedef            │ Dosya veya klasörü taşır / yeniden adlandır│ mv kaynak hedef                       │
│ dosyayaz isim.txt            │ Yeni dosya oluşturur                       │ touch isim.txt                        │
│ düzenle dosya.txt            │ Dosyayı nano ile düzenler                  │ nano dosya.txt                        │
│ temizle                      │ Terminal ekranını temizler                 │ clear                                 │
│ çık                          │ Oturumu veya terminali kapatır             │ exit                                  │
│ göster                       │ Dizin içeriğini listeler                   │ ls                                    │
│ göster -tümü                │ Gizli dosyalar dahil gösterir             │ ls -a                                 │
│ göster -uzun                │ Detaylı listeleme yapar                    │ ls -l                                 │
│ göster -tarihegöre -ters    │ Dosyaları tarihe göre ters sırayla dizer   │ ls -ltr                               │
│ göster -boyut               │ Dosyaları boyuta göre sıralar              │ ls -S                                 │
│ ağaç                         │ Klasörleri ağaç şeklinde gösterir          │ tree                                  │
│ durum                        │ Anlık sistem yükünü gösterir               │ top                                   │

🛠️ Yardımcı Komutlar:

│ bekle 5                     │ 5 saniye bekler                            │ sleep 5                               │
│ zaman                       │ Şu anki tarihi/saat bilgisi                │ date                                  │
│ arkaplan komut              │ Komutu arka planda çalıştırır              │ nohup komut &                         │

│ sistembilgi                 │ Sistem ve çekirdek bilgisi verir           │ uname -a                              │
│ yardım                      │ Bu kılavuzu gösterir                       │ cat \$HOME/kılavuz.txt                │

📌 Notlar:
- `yönetici` ifadesi `sudo` yerine kullanılmıştır.
- Bazı parametreler kısa yerine Türkçeye çevrilmiş şekliyle yazılmıştır.
- Bu sistem sadece komutları daha kolay hatırlanır hale getirir, eski komutlar da çalışmaya devam eder.

Terminali kapatıp açtıktan sonra komutları kullanabilirsiniz.
